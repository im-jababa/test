# 메모 프로그램 (테스트용)

간단한 메모 앱으로, 실행 중인 동안만 메모가 유지됩니다. 별도의 영구 저장은 없습니다.

**개요**

- 각 메모는 `id`, `제목`, `내용`, `생성 시각`, `마지막 변경 시각`을 가집니다.
- 프로그램 시작 시 메모 목록과 메모 추가 버튼이 표시됩니다.
- 메모가 존재하면 목록에서 메모를 눌러 상세 화면으로 이동해 제목과 내용을 입력/수정할 수 있습니다.
- 메모 추가 시 제목과 내용을 입력하고 생성합니다.
- 저장 버튼은 없으며, 입력 중 실시간으로 저장됩니다. 메모 추가 시에도 마찬가지입니다.
- 메모 목록에서 정렬 옵션을 선택해 `생성 시각 오름/내림`, `마지막 변경 시각 오름/내림`으로 정렬할 수 있습니다.
- 메모는 목록 화면과 메모 상세 화면에서 모두 삭제할 수 있습니다.
